PAT で設定する（所有者向け）

private リポジトリのデータを読むため、Fine-grained PAT をこの端末に保存します。 発行: GitHub → Settings → Developer settings → Fine-grained personal access tokens。 Repository access は mat-dev-io/LIFE のみ、Permissions は Contents: Read-only のみにしてください。 トークンは localStorage に保存され、GitHub API 以外へは送信されません。

共有用トークンの作成: 共有専用に発行した読み取り PAT をパスワードで暗号化します（空欄ならこの端末の PAT を使用）。 出力された JSON を assets/shared-token.json としてコミットすると、 パスワードを知っている人が閲覧できるようになります。
