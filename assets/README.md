# assets — デザイン用素材の置き場所と仕様

Apple 製品ページ風デザインを仕上げるための素材ディレクトリ。
**このリポジトリは public**。個人情報・健康データのスクリーンショット・
再配布不可ライセンスの素材（SF Symbols 等）は置かないこと。

## ディレクトリ構成

```
assets/
├── hero/    ヒーロー背景（雰囲気画像・任意で動画ループ）
├── icon/    アプリアイコン（ホーム画面追加・favicon 用）
└── og/      リンクプレビュー画像
```

## 用意する素材（優先度順）

### 1. アプリアイコン（最優先・効果大）

| ファイル | 仕様 |
|---|---|
| `icon/app-icon.png` | 1024×1024 PNG のマスター 1 枚（角丸なしのフル正方形。角丸は OS が付ける） |

- モチーフ案: 月 + リング（睡眠）や「L」モノグラム。背景は単色 or 微グラデーション
- 180/192/512px への縮小・`apple-touch-icon`・favicon・Web App Manifest の組み込みはこちらで行う
- iPhone のホーム画面に追加したときの見た目が一番変わる素材
- **生成済み派生ファイル**（マスター更新時は再生成すること）:
  `apple-touch-icon.png`(180) / `icon-192.png` / `icon-512.png` / `favicon-32.png`

### 2. ヒーロー背景（雰囲気画像）

| ファイル | 仕様 |
|---|---|
| `hero/sleep-hero.webp` | 横 1600px 程度・200KB 以下。夜空・月・深いネイビー〜藍の抽象グラデーション。**ダーク基調**（上に白文字が載る前提） |
| `hero/activity-hero.webp` | 同仕様。夜明け・朝の光・動きを感じる抽象。こちらもダーク基調に寄せると両モードで破綻しない |
| `hero/screen-hero.webp` | 同仕様。暗い部屋で画面だけが光っている抽象（下寄りの青白い光・周縁に紫・芯にアンバー） |

- 実寸は 3 枚とも **1600×679**（比 2.36:1）。`center / cover` で表示されるため、見せ場は中央帯に置き四隅に情報を持たせない
- 写真より**抽象グラデーション/ブラー**の方が文字と干渉せず Apple らしくなる
- ライトモード用に別画像を分けたい場合は `-light` / `-dark` サフィックスで 2 枚ずつ
- 入手先: Unsplash / Pexels（ライセンス上再配布可）、または AI 生成（個人サイトなので可）。CC0 推奨
- **AI 生成画像は右下に透かし（星マーク）が入ることがある**。周囲の背景色で塗り潰し、
  フェザーマスクを掛けたぼかしで継ぎ目を消してから配置する（PR #10 / #13 で実施）

### 3. OG 画像（リンクプレビュー）

| ファイル | 仕様 |
|---|---|
| `og/og-image.jpg` | 1200×630 JPEG。「LIFE」ワードマーク + タグライン程度のシンプルな構成（写真調のグラデーションは PNG だと肥大するため JPEG） |

### 4. 任意（なくても実装可能）

- **統計ラベル用アイコン**: `icon/` に SVG（月・ベッド・心拍・炎・足跡など）。
  [Lucide](https://lucide.dev) / [Phosphor](https://phosphoricons.com)（MIT）から選ぶ。
  ※用意がなければ実装時に Lucide からインライン SVG で組み込むので必須ではない
- **ヒーロー動画ループ**: `hero/sleep-loop.mp4`。6〜10 秒・無音・H.264・2MB 以下 +
  ポスター画像。ゆっくり流れる夜空など。スマホ回線を考えると画像で十分なので後回しで可

## 禁止事項

- 個人が特定できる写真・実データのスクリーンショット（public リポジトリのため）
- SF Symbols のアセット同梱（ライセンス上 Apple プラットフォーム外での利用不可）
- 1 ファイル 500KB 超の画像（スマホ回線での初回表示が重くなる）
